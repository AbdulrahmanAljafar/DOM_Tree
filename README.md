# HTML Tree

## Features:-
- Visualize the dom nodes
- Visualize text nodes
- ( Expand / Collapse ) nodes
- Display node attributes
- Display node outer html
- Export Visualization to jpg
- Ability to add nodes
- Ability to Edit text
- Ability to move nodes
